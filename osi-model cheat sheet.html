OSI Model Cheat Sheet

The 7 OSI Layers

Layer Name Purpose Example Cybersecurity Relevance
7 Application User-facing apps (HTTP, FTP, email). Web browsers, Gmail, APIs. Phishing, malware downloads, API attacks.
6 Presentation Data formatting (encryption, compression). SSL/TLS, JPEG/MPEG encoding. Decrypting malicious traffic, data tampering.
5 Session Manages connections between devices. VPNs, NetBIOS. Session hijacking, brute-force attacks.
4 Transport End-to-end communication (TCP/UDP). Port numbers, flow control. Detecting port scans, DDoS attacks.
3 Network Routing data between networks (IP addresses). Routers, IP packets. Blocking malicious IPs, subnet monitoring.
2 Data Link Direct communication between devices (MAC). Switches, Ethernet frames. MAC spoofing, ARP poisoning.
1 Physical Physical hardware (cables, Wi-Fi signals). Fiber optics, hubs, NICs. Cable tampering, rogue access points.

How to Memorize

Mnemonic: "All People Seem To Need Data Processing"

  • Application → All
  • Presentation → People
  • Session → Seem
  • Transport → To
  • Network → Need
  • Data Link → Data
  • Physical → Processing

Example Attack Flow

Phishing Attack:

  1. Layer 7: Malicious email sent via SMTP.
  2. Layer 3: Attacker’s IP logged in firewall.
  3. Layer 4: Suspicious TCP traffic on port 80.
  4. SOC Action: Block IP at Layer 3, analyze logs at Layer 7.
